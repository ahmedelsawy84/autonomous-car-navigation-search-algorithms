# Autonomous Car Navigation Using Search Algorithms

This project demonstrates graph-based autonomous car navigation using Python search algorithms.

## Project Overview

The project applies search algorithms to route-finding and graph traversal problems. It includes:

- Uniform Cost Search for minimum-cost pathfinding
- Breadth-First Search traversal with step-by-step visualization
- Greedy Best-First Search using heuristic values
- A* search implementation

## Technologies Used

- Python
- NetworkX
- Matplotlib
- heapq
